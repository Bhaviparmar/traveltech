== Description ==

Yala Travel is an Awesome Woocommerce based Travel & Trekking WordPress Theme, It Consist of Different Section which can easily managed by Elementor Drag and Drop Features.It have banner section, Blog Section, Destination Secions, Popular Trips Section, Service Section. View Demo https://yalathemes.com/demo/


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
